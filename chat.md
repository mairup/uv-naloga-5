For this assignment, you should select the **Empty Views Activity** template (sometimes just called **Empty Activity** in older versions of IntelliJ IDEA).

Here is why this is the exact template you need, along with a critical warning about newer IDE versions:

### The "Views" vs "Compose" Trap

The assignment explicitly requires you to use classic Android UI components like `Spinner`, `ListView`, and `RecyclerView`. These rely on traditional **XML layouts**.

In recent versions of IntelliJ and Android Studio, the default "Empty Activity" template uses _Jetpack Compose_, which is a modern UI framework that does not use XML or these specific components. **Do not choose Jetpack Compose**, or your implementation won't align with the assignment's grading criteria (like the extra points for using a `RecyclerView`).

### Project Setup Checklist

When creating the project, configure it like this:

- **Template:** Empty Views Activity
- **Language:** Kotlin or Java (Check your course syllabus, but Kotlin is the modern standard).
- **Minimum SDK:** API 24 (Nougat) is usually a safe default unless your professor specified otherwise.
- **Build configuration language:** Gradle Kotlin DSL or Groovy DSL (either is fine, Groovy is more traditional).

### How to Structure Your App

Once the template generates your project, you will have a single `MainActivity`. Based on the assignment text, you will need to manually add two more activities:

1.  **`MainActivity` (Generated by template):**
    - Will hold your `Spinner`/`RecyclerView` and the five main buttons (**Dodaj vnos**, **Popravi izbranega**, **Izbriši izbranega**, **Izračunaj ITM**, **Izračunaj dozo**).

2.  **`PersonActivity` (You will create this):**
    - This is the **"nova aktivnost"** for the form to input or edit the user's details (Name, Surname, Gender, DOB, Weight, Height).
    - _Tip:_ You can use this exact same activity for both the "Dodaj" (Add) and "Popravi" (Edit) flows by passing different Intents.

3.  **`DosageActivity` (You will create this):**
    - This is the **"sekundarno aktivnost"** for the medication math.
    - _Tip:_ Because this activity needs to return the calculated dosage back to the main screen, you should look into using `ActivityResultLauncher` (the modern replacement for `startActivityForResult`) to pass the data back to `MainActivity`.
